What we defend against

Real-world threats. Real CVEs. Real protection.

Every rule is derived from documented OpenClaw vulnerabilities, MITRE ATLAS techniques, and public security audits.

WebSocket Hijack → RCE

CVE-2026-25253 · CVSS 8.8

Cross-origin WebSocket hijacking that steals gateway tokens and executes arbitrary code with user privileges.

rcewebsocket

OS Command Injection

CVE-2026-25157 · CVE-2026-24763

SSH path injection and Docker PATH manipulation that chains OS commands to gain remote code execution.

sshdocker

Prompt Injection

Classic "ignore previous instructions" attacks, DAN jailbreaks, and persona override attempts that bypass system prompts.

jailbreakoverride

Credential Exfiltration

Detects and redacts API keys (Anthropic, OpenAI, GitHub, AWS) leaked through tool calls or model outputs. Found in 40% of public configs.

secretsredact

Privilege Escalation

Blocks low-trust sessions (group chats, DMs) from executing destructive commands or sending messages to the operator session.

escalationsession

Supply Chain & 0-Click RCE

Catches obfuscated payloads from poisoned skills, base64-encoded eval, and email-triggered remote code execution via Gmail hooks.

supply-chaingmail
Architecture

Transparent proxy. Zero code changes.

Agent Guard sits between your agent and the LLM provider. It inspects every request and response in real time.

1

Intercept

Your agent sends API calls to Agent Guard instead of directly to the provider.

→
2

Inspect

Policy rules evaluate tool calls, message content, session tier, and secrets in microseconds.

→
3

Decide

Allow, Block, or Redact — based on your YAML policy, with full audit logs.

→
4

Forward

Safe requests are streamed to the LLM provider with <5ms overhead. Responses flow back transparently.
